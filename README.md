# w_prime

App criado para estudos usando de referência o Netflix e o HBO Max, consumindo a API do https://themoviedb.org/

Utilizado:
- Jetpack Compose
- Koin
- Coil
- Refrofit
- MVVM
- LiveData
- Coroutines

Será Utilizado: 

- Room Database

Será utilizado nos testes unitários e automatizados:

- JUnit
- Espresso
